# Store Submission Checklist

- [ ] App name, subtitle, short description 확정
- [ ] Privacy policy URL 연결
- [ ] iOS App Store screenshots (6.7", 6.5", iPad)
- [ ] Android Play Store screenshots (phone, tablet)
- [ ] App icon 1024x1024 최종 교체
- [ ] Splash/background branding 최종 교체
- [ ] Android signing key / iOS certificates 확인
- [ ] `eas.json` submit credentials 실제 값 반영
- [ ] `app.json` bundle/package id 최종 검토
- [ ] 앱 심사 노트(오프라인 자산 포함, 네트워크 비의존) 작성
